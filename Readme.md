# Wallpaper Switcher

A simple PowerShell script that changes your desktop wallpaper.

## Features

- Switches between multiple wallpapers
- Remembers the last wallpaper used
- Can be started with a keyboard shortcut

## Installation

1. Put the script here:

C:\Backgrounds\wallpaper.ps1

2. Put your wallpaper images in the same folder.

3. Edit the image paths in the script if needed.

## Run the Script

Run with PowerShell:

powershell.exe -ExecutionPolicy Bypass -File "C:\Backgrounds\toggle.ps1"

## Keyboard Shortcut

Create a shortcut to the script:

1. Right-click the shortcut
2. Select Properties
3. Select Shortcut key
4. Press:

Ctrl + Alt + W

## Files

- toggle.ps1 - PowerShell script
- state.txt - stores the current wallpaper state
- README.md - this file
